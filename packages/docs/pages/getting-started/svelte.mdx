# Svelte

GridSheet provides a comprehensive Svelte component that brings Excel-like functionality to your Svelte applications.

## Installation

```bash
npm install @gridsheet/svelte-core
# or
yarn add @gridsheet/svelte-core
# or
pnpm add @gridsheet/svelte-core
```

## Basic Usage

```svelte filename="App.svelte" copy
<script>
  import { GridSheet, buildInitialCellsFromOrigin } from '@gridsheet/svelte-core';
  
  const initialCells = buildInitialCellsFromOrigin({
    matrix: [
      ['Name', 'Age', 'City'],
      ['John', 25, 'New York'],
      ['Jane', 30, 'Los Angeles'],
      ['Bob', 35, 'Chicago']
    ]
  });
  
  const options = {
    sheetHeight: 400,
    sheetWidth: 600,
    sheetResize: 'both'
  };
</script>

<GridSheet 
  {initialCells} 
  {options}
  sheetName="MySheet"
/>
```

**Note:** This Svelte implementation is a wrapped version of the Preact core, which means some advanced features may be limited. Additionally, while GridSheet supports custom renderers for arbitrary components, Svelte components cannot be passed directly to GridSheet. For custom component rendering, you need to write the code in Preact and pass it to GridSheet.
