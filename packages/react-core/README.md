## Introduction

@gridsheet/react-core is a simple yet highly functional spreadsheet component for ReactJS.

![gridsheet](https://github.com/walkframe/gridsheet/raw/master/gridsheet.png)


## Installation

```sh
$ npm install @gridsheet/react-core --save
```
